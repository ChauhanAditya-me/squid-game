# Frontend Folder

This folder contains all files for the web client:

- `index.html`: Main HTML file
- `style.css`: CSS styles
- `script.js`: Main game logic
- `assets/`: Audio files (see README inside)
- `dev/`: Development/debug scripts
- `tests/`: Test tools (e.g., backend API tester)

See `docs/README.md` for full project details.
